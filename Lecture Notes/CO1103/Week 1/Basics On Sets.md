## Notes:

- What is a set? 
	- A collection of objects where
		- order does not matter
		- repetition does not matter

- Curly brackets mark the start and end of a set.

- We can give names to sets.
	- eg: V = {a,e,i,o,u}

### Set Membership

eg : 

```python
3 ∈ {1,3,5,7}

3 is a member of the set
```

Negation of set membership:

```python
4 ∉ {1,3,5,7}

4 is not a member of the set
```

### Cardinality of a set

- |A| the number of distinct elements of A

Examples: 
```python
|{0,1}| = 2

```
