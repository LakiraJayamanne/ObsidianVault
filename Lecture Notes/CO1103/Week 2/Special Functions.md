# Polynomial 

 A polynomial function degree *n* is a function f: R -> R  defined by an algebraic expression of the form: 
 
$$
f(x) = a_n x^n + a_{n-1}x^{n-1} + \dots + a_1 x + a_0
$$

where n ∈ N
